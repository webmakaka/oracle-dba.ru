# Oracle Database Sheduler

Sheduler - помогает автоматизировать задания внутри базы данных Oracle. Состоит из
пяти базовых компонентов:

<ul>
<li>заданий (jobs)</li>
<li>расписаний (shedules)</li>
<li>программ</li>
<li>событий</li>
<li>цепочек</li>
</ul>
